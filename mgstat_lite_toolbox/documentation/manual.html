 variable name dimensions class description
input functionname 1xN char Char array referring to an existing function, either one of the mgstat-lite toolbox, or one of MATLAB's.

If "functionname" refers to an m-file of the mGstat-lite toolbox, the help documentation of that mGstat-specific file is opened in the MATLAB help browser.

If the requested "functionname" is a MATLAB function, then "doc" is called with the function name specified.

Examples of how the command "manual" can be used:

manual svgram_exp
manual('svgram_exp')
manual plot



Example of how the command "manual" should not be used:

manual(svgram_exp)

manual('contents')

Opens an overview of the available help documentation specific to the mGstat_lite toolbox.

Equivalent to:

manual

manual

Just typing the word "manual" at the prompt will invoke an overview of the contents of the mGstat-lite toolbox.

Equivalent to:

manual('contents')

See also:
doc MATLAB function "doc".
help MATLAB function "help".
contents mGstat-lite toolbox contents
