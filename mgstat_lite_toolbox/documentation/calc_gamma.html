calc_gamma

gamm = calc_gamma(svmodel, lag_vec)

gamm = calc_gamma(svmodel, lag_vec)

input/
output
 variable name dimensions class description
input svmodel 1x1 cell cell Cell array containing information on the semivariogram model. Format is:
svmodel{1,1}.c0 % nugget
svmodel{1,1}.c1 % c1
svmodel{1,1}.type % type of semivariogram model: 'Exp', 'Gau', or 'Sph'
svmodel{1,1}.range % range
svmodel{1,1}.longname % long identifier
svmodel{1,1}.shortname % short identifier used in gstat.exe command file (should be one word)



input lag_vec 1xN numeric Vector of lag values.
output gamm 1xN numeric Semivariance or cross-semivariance at lags "lag_vec" calculated according to the semivariogram model supplied in "svmodel".

Calculates the semivariance for the lag entries specified in variable "lag_vec" using a semivariogram model, the type and parameters of which must be specified in a cell array ("svmodel"). See example below:

lag_vec = linspace(0,50,1000)';

svmodel{1,1}.c0 = 4;
svmodel{1,1}.c1 = 18;
svmodel{1,1}.type = 'Gau';
svmodel{1,1}.range = 25;

g = calc_gamma(svmodel,lag_vec);

plot(lag_vec,g, 'b-')
xlabel('lag')
ylabel('semivariance')
grid on
