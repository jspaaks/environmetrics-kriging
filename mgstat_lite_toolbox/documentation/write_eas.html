write_eas(filename,data);

Writes the columns of "data" to *.eas file "filename".

Example:

x = [8;2;7;4;3;6;9;5;1]
y = [5;6;9;1;4;2;8;3;7]
v = [1.4; 8.2; 4.3; 8.9; 7.3; 6.9; 3.5; 1.7; 1.6]

pos_known = [x,y];
val_known = v;

write_eas('obs.eas',[pos_known val_known]);

Generates a file that looks like this:

write_eas(filename,data,headercell);

Writes the columns of "data" to *.eas file "filename" and adds metadata from "headercell" to the first lines of the GeoEAS file.

Example:

x = [8;2;7;4;3;6;9;5;1]
y = [5;6;9;1;4;2;8;3;7]
v = [1.4; 8.2; 4.3; 8.9; 7.3; 6.9; 3.5; 1.7; 1.6]
headercell = {'x-coordinate','y-coordinate','observed value'}

pos_known = [x,y];
val_known = v;

write_eas('obs.eas',[pos_known val_known], headercell);

Generates a file that looks like this:

write_eas(filename,data,headercell);

Writes the columns of "data" to *.eas file "filename" using the data format specified in "format_str"; and adds metadata from "headercell" to the first lines of the GeoEAS file.

Example:

x = [8;2;7;4;3;6;9;5;1]
y = [5;6;9;1;4;2;8;3;7]
v = [1.4; 8.2; 4.3; 8.9; 7.3; 6.9; 3.5; 1.7; 1.6]
headercell = {'x-coordinate','y-coordinate','observed value'}

pos_known = [x,y];
val_known = v;

write_eas('obs03.eas',[pos_known val_known], headercell, '%4u%2u%7.2f');
