addlabels

addlabels(lag,gamm,count)

addlabels(lag,gamm,count)

input/
output
 variable name dimensions class description
input lag 1xN numeric Vector with lag value for the N bins.
input gamm 1xN numeric Vector with semivariance value for the N bins.
input count 1xN numeric Vector with number of paired observation for the each of the N bins.

Adds labels to the bins of the semivariogram with the number of paired observations. For example, we may have an array with observation locations "pos" and the value of the variable of interest at that location "val". The semivariance array, lag array and count array may be calculated by the function svgram_exp and plotted according to:

[lag,gamm,count] = svgram_exp(pos,val,'cutoff',1200,'width',100);
figure('position',[51 119 651 400])
subplot('position', [0.14, 0.14, 0.77, 0.75])

plot(lag,gamm,'+k','markersize',5)

set(gca,'tickdir','out',...
'xtick', lag+lag(1),...
'xticklabel', [],...
'yticklabel', [],...
'ylim', [0,35000],...
'xlim', [0,1200],...
'xgrid', 'on',...
'fontsize', 10)

xlabel('\ith','fontsize',14)
ylabel('\it\gamma (h)','fontsize',14)

reformat_ticks(gca, 'x','%8.0f')
reformat_ticks(gca, 'y','%8.0f')

Inclusion of the "addlabels" command will display the number of paired observations in the correct bin of the figure:

addlabels(lag,gamm,count)
