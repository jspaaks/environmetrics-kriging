Contents of the mGstat-lite toolbox

addlabels adds labels to bins of variogram
calc_gamma calculates the semivariance from the semivariance model
causch tests whether the Cauchy-Schwartz condition is met
gstat_binary returns the location of the MS-DOS gstat executable
imagesc_plot combined visualization of vector and matrix data
isint determines whether the input array is integer or not
manual invokes the mGstat-lite and MATLAB help documentation
mgstat_lite translates the kriging options into command files and runs "gstat.exe".
read_eas reads GeoEAS (*.eas) formatted file
reformat_ticks reformats the tick labels of plots
sv_exponential exponential semivariogram model
sv_gaussian Gaussian semivariogram model
sv_nugget nugget semivariogram model
sv_spherical spherical semivariogram model
svgram_exp calculation of the semivariogram
write_eas writes GeoEAS (*.eas) formatted file
