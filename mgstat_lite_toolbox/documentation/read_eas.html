input

output
 data NxM double data read from file "filename".

Reads the columns of the specified "filename" into the array "data". The first lines of the GeoEAS file are ignored.

[data, header] = read_eas(filename)

input/
output
 variable name dimensions class description
input
 filename 1xN char String containing the name file from where the data must be read.
output
 data NxM double data read from file "filename".
output
 header 1xN cell column descriptors read from file "filename".

Reads the columns of the specified "filename" into the array "data". The first line of the GeoEAS file is ignored, but the column descriptors are read into the cell array "header".
