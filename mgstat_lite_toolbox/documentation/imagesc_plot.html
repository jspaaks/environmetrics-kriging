imagesc_plot

imagesc_plot(M, xvec, yvec, x_ax_lims, y_ax_lims)
[h01,h02] = imagesc_plot(M, xvec, yvec, x_ax_lims, y_ax_lims)

imagesc_plot(M, xvec, yvec, x_ax_lims, y_ax_lims)

input/
output
 variable name dimensions class description
input M NxM numeric 2-D array.
input xvec 1xN numeric Vector with x coordinates of observations.
input yvec 1xN numeric Vector with y coordinates of observations.
input x_ax_lims 1x3 numeric Vector with x axis limtis and interval.
input y_ax_lims 1x3 numeric Vector with y axis limtis and interval.


This function can be used to visualize a 2-D array ("M") and 1-D arrays ("xvec" and "yvec") in the same subplot, which is useful for indicating the locations of measurements (i.e. vector data) on top of a map (i.e. 2-D array). Normally, the two arrays can only be visualized separately, see example below.

This code:

subplot(2,1,1)
imagesc(M)
axis image
set(gca,'ydir', 'normal', 'xlim', [0,141], 'ylim', [0,101])

subplot(2,1,2)
plot(xvec,yvec,'k.')
axis image
set(gca,'xlim',[-5,65], 'ylim', [-5,45])

will result in the figure below (note that the direction of the y-axis has been set to 'normal' to be consistent with the plot function's y-axis direction):





In contrast, "imagesc_plot" visualizes the 2-D array "M" and subsequently overlays this map with the vector data from "xvec" and "yvec". In order to arrange them properly, the x-axis limits and y-axis limits must be specified in "x_ax_lims" and "y_ax_lims". The format of "x_ax_lims" must be: x-coordinate of the center of the last column of "M", interval in units of x, x-coordinate of the center of the last column of "M". A similar format applies to the y-axis. See code below:

imagesc_plot(M,xvec,yvec, [-5,10,65],[-5,10,45])



will result in a figure like the one below:



[h01,h02] = imagesc_plot(M, xvec, yvec, x_ax_lims, y_ax_lims)

input/
output
 variable name dimensions class description
input


same as above
output h01 1x1 double Handle to the axis of the imagesc command with which M is visualized.
output h02 1x1 double Handle to the axis of the plot command with which [xvec,yvec] are visualized.

Same as "imagesc_plot(M, xvec, yvec, x_ax_lims, y_ax_lims)", but returns handles to the axes of the "imagesc" and "plot" commands.
