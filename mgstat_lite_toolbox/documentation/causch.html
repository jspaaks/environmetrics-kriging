output
 variable name dimensions class description
input svmodel 2x1 cell cell Cell array containing information on the semivariogram model. Format is:
svmodel{α,1}.c0 % nugget
svmodel{α,1}.c1 % c1
svmodel{α,1}.type % type of semivariogram model: 'Exp', 'Gau', or 'Sph'
svmodel{α,1}.range % range
svmodel{α,1}.longname % long identifier
svmodel{α,1}.shortname % short identifier used in gstat.exe command file (should be one word)

(Where α is either 1 or 2)

input xvmodel 1x1 cell cell Cell array containing information on the cross-variogram model. Format same as "svmodel".
input lag_vec 1xN numeric Vector of lag values for which Cauchy-Schwartz is checked.
output cond 1x1 logical logical(1) if Cauchy-Schwartz condition is true, logical(0) otherwise.

Tests whether the semivariogram models (variable "svmodel") and cross-variogram model (variable "xvmodel") together are consistent with the Cauchy-Schwartz condition over the lag range "lag_vec". The function returns logical(1) if Cauchy-Schwartz condition is true, logical(0) otherwise.
