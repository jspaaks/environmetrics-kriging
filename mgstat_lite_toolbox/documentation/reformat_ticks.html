reformat_ticks

reformat_ticks(CurAxes, Ax, formatstr)

input/
output
 variable name dimensions class description
input CurAxes 1x1 double handle to the current axis. See also MATLAB's function "gca".
input Ax 1xN char identifies the axis whose ticklabels must be reformatted according to "formatstr". Choose from 'x' or 'y'.
input formatstr 1xN char char array according to which the axis must be formatted.

Formats the tick labels on axis "Ax" in axes "CurAxes" according to format "formatstr".

For example, the tick labels in the first subplot of the figure below have been reformatted using the value '%8.1f' as format string, resulting in the tick marks of the second subplot.
